# 预装软件与环境清单

这份清单给准备新电脑的人使用。核心原则：成熟桌面软件和账号权限提前准备；
医学文献追踪系统、Hermes、CC Switch、uv、隔离 Python 由完整安装包自动部署。

## 必须准备

| 项目 | 要求 | 用途 | 官方入口 |
| --- | --- | --- | --- |
| Windows | Windows 10 或 Windows 11，64 位优先 | 运行一键安装器和计划任务 | 系统自带 |
| 网络 | 能访问 GitHub、DeepSeek、PubMed、Europe PMC、medRxiv、ClinicalTrials.gov、飞书 | 下载、模型调用、文献 API 和消息投递 | 由本机网络决定 |
| 磁盘空间 | 至少 5 GB 可用空间 | 安装 Hermes、隔离 Python、依赖和保存文献数据 | 本机磁盘 |
| GitHub 账号 | 对本仓库有访问权限 | 下载私有 Release 安装包 | https://github.com |
| DeepSeek API Key | 能调用 DeepSeek API | Hermes 使用 `deepseek-v4-pro` 生成摘要 | https://platform.deepseek.com/api_keys |
| 飞书账号 | 能扫码登录，并有创建或安装企业自建应用的权限 | 创建机器人并接收每日推送 | https://www.feishu.cn/download |

## 建议提前安装

| 软件 | 是否必须 | 说明 | 官方入口 |
| --- | --- | --- | --- |
| 飞书桌面端 | 强烈建议 | 安装器会通过 Hermes 配置飞书机器人；桌面端方便扫码、授权和接收消息 | https://www.feishu.cn/download |
| Obsidian | 可选 | 仅用于后续人工整理文献笔记，不影响自动追踪和飞书推送 | https://obsidian.md/download |
| 常用浏览器 | 建议 | 飞书开放平台授权、DeepSeek 控制台和 GitHub 下载都需要浏览器 | Edge/Chrome 等 |

## 不需要提前安装

完整 Windows 包会自动处理以下组件，普通用户不需要预装：

- Python
- Git
- Node.js
- Hermes Agent
- CC Switch
- uv
- Codex
- OpenClaw
- WSL2

其中 CC Switch 会以便携版安装到 `%LOCALAPPDATA%\CCSwitchPortable`，用于配置管理；
医学文献定时追踪运行时仍只依赖 Hermes Agent，不调用 Codex 或其他 agent。

## 可选账号与 API

| 项目 | 何时需要 | 说明 |
| --- | --- | --- |
| NCBI API Key | PubMed 查询量变大时 | 当前系统没有强制要求；后续多主题、高频率运行时可配置以提高 NCBI 请求额度 |
| NCBI Email | 更规范地调用 NCBI E-utilities 时 | 可作为环境变量配置，方便 NCBI 联系维护者 |
| GitHub Desktop 或 GitHub CLI | 开发者维护时 | 普通安装用户不需要；只有要改代码、发版或调试 GitHub 流程时才需要 |

## 安装包选择规则

新电脑从最新 Release 下载：

https://github.com/yrupeechalco-cell/medical-literature-tracker-hermes/releases/latest

下载文件名包含 `full-windows` 的 ZIP，例如：

`medical-literature-tracker-full-windows-0.3.1.zip`

不要把以下文件当作新电脑安装包：

- `medical-literature-tracker-hermes-0.3.1.zip`：轻量源码包。
- GitHub 自动生成的 `Source code (zip)`。
- GitHub 自动生成的 `Source code (tar.gz)`。

## 安装前最终检查

1. 已登录有仓库权限的 GitHub 账号。
2. 已下载 `full-windows` ZIP。
3. ZIP 来自 GitHub Release 页面，不是聊天附件或第三方网盘。
4. DeepSeek API Key 已准备好。
5. 飞书账号可扫码并有应用授权权限。
6. 电脑至少有 5 GB 空闲空间。
7. 网络能打开 GitHub、DeepSeek、飞书和 PubMed。

通过以上检查后，解压完整 ZIP，双击 `INSTALL_WINDOWS.cmd`。
