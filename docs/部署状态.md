## 已知边界

- medRxiv 在本次 14 天窗口内没有符合主题的预印本，这不是采集错误。
- 第一次运行属于回填，会比后续日常更新多。
- 上游 API 会因 1 天重叠窗口再次返回少量旧记录，但 SQLite 会在生成简报前去重；
  这是为了捕获延迟索引或版本更新，不会重复推送已交付版本。
- 每个采集运行最多生成一个简报批次，候选仅限该次运行中新建或版本变化的记录；
  当天未入选的低优先级旧记录不会在未来日期形成历史补推队列。
- 摘要来自数据库公开元数据，不能替代阅读全文和人工证据质量评价。
- 当前未抓取付费全文，避免版权和许可风险。
- Hermes gateway 当前为手动后台进程，不是 Windows 服务；重启电脑后可能需要重新启动 gateway。
